Nintendo Game Search:



Want to find some Wii, Wii U games, and more along with great features as well?


For Wii and Wii U Games, I strongly recommend using GameTDB, a game database.


Although 3DS and DS Game sections aren't on the site yet, you get awesome stuff like this:


  • An awesome search for games! The Wii version is available here and the Wii U version is available here.
  • A downloads page hosting weekly-made cover packs! The Wii version is available here and the Wii U version is available here.
  • A list of all games! The Wii version is available here and the Wii U version is available here. A text file with the same list is updated weekly and is in the "Downloads" section.
  • Covers, discs, and more types of artwork! There's 3 quality types available for front covers (Wii U version only at this time) - Regular, M (Medium), and HQ (High Quality).
  • Detailed game information! Release dates, game ratings, game descriptions, and more!
