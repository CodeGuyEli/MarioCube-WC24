Wii 리모컨 사용 방법
포인터 가리키는 방향으로 커서()를 움직인다.
(포인트하기)

● 포인트한 Mii 또는 항목을 선택한다.

● 별자리판(→P.5: 메인 화면 설명)을 잡는다.(잡은 상태로 커서를 움직이면 별자리판을 돌릴 수 있습니다.)

● 행운의 컬러를 확인한다.
열린 창을 닫는다.
＋

● Mii를 잡는다.

● (길게 눌러서) 행운의 컬러를 확인한다.
Mii를 전환한다.
HOME 버튼 메뉴를 표시한다.

● Wii 리모컨을 사용하기 전에 스트랩에 이상이 없는지 점검해 주십시오. 만약 스트랩에 이상이 있을 경우에는 Wii 리모컨의 사용을 중지하고 반드시 스트랩을 교체해 주십시오.

주의

● Wii 리모컨을 보다 안전하게 사용하기 위해 Wii 리모컨 커버를 씌워서 사용하기를 권장합니다.
