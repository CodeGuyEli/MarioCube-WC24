7
Le taux de compatibilité

Cette fonction vous permet de vérifier le taux de compatibilité entre différents Mii. Pourquoi ne pas vérifier à quel point vous êtes compatible avec quelqu’un d’autre avant de travailler ou de sortir ensemble?

Note: vous devez avoir ajouté au moins deux Mii dans la chaîne pour utiliser cette fonction. Vous pouvez vérifier la compatibilité de 6 Mii maximum. Le taux de compatibilité varie selon le nombre et les combinaisons de Mii.

  1. Sélectionner l’icône de compatibilité

    A l’écran principal, sélectionnez l’icône (voir la section 5 “L’écran principal”).

  1. Choisir les Mii

    Attrapez les Mii pour lesquels vous voulez obtenir le taux de compatibilité et déposez-les dans l’icône . Pour attraper un Mii, pointez-le et maintenez et enfoncés. Pour retirer des Mii, attrapez-les dans l’icône et déposez-les sur la roue de la chance. Sélectionnez OK! pour confirmer votre choix de Mii.

Note: vous pouvez aussi directement déposer les Mii pour lesquels vous voulez obtenir le taux de compatibilité dans l’icône à l’écran principal.
  1. Evaluation du taux de compatibilité

    Une fois que vous avez sélectionné OK!, une photo de tous les Mii participant à l’évaluation apparaît à l’écran. Cette photo représente la compatibilité de ces Mii pour la journée en cours. Si cette journée n’est pas faste, l’option VOIR LE PROCHAIN JOUR FASTE s’affiche. Sélectionnez cette icône pour voir quelle sera la prochaine journée faste pour les Mii sélectionnés.

    Lorsque vous avez terminé, sélectionnez OK pour retourner à l’écran principal.
